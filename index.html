Images are stored on Cloudinary (free, 25 GB) and the image list is synced via JSONBin.io (free) — so photos appear on every device automatically. No server needed.

One-time setup — 5 minutes

1
Cloudinary: Sign up free → Dashboard → copy your Cloud Name → Settings → Upload → Add preset → set Unsigned → copy preset name
2
JSONBin: Sign up free at jsonbin.io → click + New Bin → paste {} as content → Save → copy the Bin ID from the URL
3
JSONBin API Key: In JSONBin → go to API Keys in the menu → create a key → copy it
4
Paste all four values below → Save & Connect. Done! Same credentials on every device = shared gallery.
